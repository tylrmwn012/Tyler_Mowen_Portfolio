This personal project focused on modeling the mechanical motion of an F-35Bâ€“inspired swivel nozzle using four independently articulated sections, each monitored by its own rotary encoder. The goal of the project was to study how multiple degrees of freedom can be coordinated to achieve precise and repeatable motion in a complex mechanical system. Rather than simulating airflow or thrust, the project emphasized position sensing, synchronization, and control logic to replicate the mechanical behavior of a thrust-vectoring nozzle.

The nozzle was designed with four distinct points of articulation, allowing each section to rotate through a defined angular range. A dedicated rotary encoder was assigned to each joint, providing continuous feedback on angular position. This required careful mechanical alignment and calibration to ensure smooth and reliable motion while preventing interference between adjacent sections. The multi-encoder configuration enabled fine-grained control of nozzle geometry and realistic modeling of staged nozzle articulation.

Control logic was implemented to read encoder data from all four articulation points and coordinate their motion as a unified system. Encoder feedback was used to verify that each section reached its intended position and remained synchronized with the others, demonstrating the importance of closed-loop control in multi-axis mechanical systems. This approach helped prevent cumulative positioning errors and ensured consistent, repeatable motion across the entire mechanism.

Overall, this project was intentionally designed as a foundation for future expansion. A key planned improvement is to implement a higher-level control interface that allows a user to simply input a desired output angle in degrees, with the system automatically calculating and commanding the required positions for each articulated section. This enhancement would further abstract the control logic, improve usability, and move the project closer to a realistic thrust-vectoring control system, while continuing to build skills in embedded control and system integration.
