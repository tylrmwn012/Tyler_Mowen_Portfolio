CSC Messaging Application Project Timeframe Winter 2025 - Spring 2025 Overview OVERVIEW GOES HERE Tools and Techniques GO HERE GO HERE GO HERE Skills Learned Flutter / Dart Programming Google Firebase Web Socket Programming

During my internship with the Center for Supportive Communities (CSC), I contributed to the development of a messaging application designed to support communication between students, mentors, teachers, and staff. The purpose of this work was to help create a reliable and accessible platform that could facilitate timely interaction within CSC’s educational support programs. This experience provided exposure to developing software intended for real-world use, with an emphasis on usability, reliability, and secure data handling.

The application was developed using Flutter and Dart to enable cross-platform deployment across mobile devices. Firebase served as the backend infrastructure, providing user authentication, real-time messaging, and cloud-based data storage. My role focused primarily on the messaging component of the application, including managing authentication states, structuring message data, and ensuring real-time synchronization across users. This required careful coordination between frontend logic and backend services to maintain consistency and performance.

A major component of my work involved implementing real-time communication using Firebase’s data services. Messages were delivered and updated instantly through active listeners, allowing users to engage in seamless conversations. The system was designed to handle multiple user roles and conversation threads while enforcing appropriate access controls. Special attention was given to error handling and data integrity to ensure stable performance under varying network conditions.

Overall, this internship project strengthened my skills in mobile application development and real-time system design. Working on a production-level application for CSC highlighted the importance of scalable architecture, security considerations, and user-centered design. The experience enhanced my proficiency in Flutter, backend integration, and collaborative development, and it reinforced my interest in building technology that has a direct and positive impact on communities.
