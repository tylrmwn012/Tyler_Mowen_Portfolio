Ultrasonic Radar Project Timeframe Summer 2025 Overview OVERVIEW GOES HERE Tools and Technique GO HERE GO HERE GO HERE Skills Learned SKILLS GO HERE SKILLS GO HERE SKILLS GO HERE

This personal project involved designing and building an ultrasonic radar system using an Arduino microcontroller to explore object detection and spatial scanning techniques. The goal was to gain hands-on experience with embedded systems while also integrating software-based data visualization. Instead of traditional radar technology, the system relied on ultrasonic sound waves to measure distance, with real-time data transmitted from the Arduino to a computer for graphical display.

The hardware setup consisted of an Arduino Uno, an HC-SR04 ultrasonic distance sensor, and a servo motor. The ultrasonic sensor was mounted on the servo motor, allowing it to rotate horizontally and scan across a range of angles. The servo motor was controlled through pulse-width modulation signals from the Arduino, while the ultrasonic sensor’s trigger and echo pins were connected to digital input and output pins. This configuration enabled consistent angular scanning and distance measurements across the sensor’s field of view.

The software component of the project was divided between the Arduino and a Python-based visualization program. The Arduino controlled the servo motion, triggered the ultrasonic sensor, calculated distance measurements using time-of-flight data, and sent angle and distance values over serial communication. A Python script running on the computer continuously read this data and used the Matplotlib library to render a radar-style display. The visualization plotted distance as a function of angle on a polar coordinate system, dynamically updating as new data was received. Detected objects appeared as points or arcs on the radar screen, providing an intuitive real-time representation of the surrounding environment.

Overall, the integration of Python and Matplotlib significantly enhanced the project by transforming raw sensor data into a clear and interactive visual output. While some noise and jitter were visible on the radar screen due to ultrasonic sensor limitations and servo movement, the display effectively conveyed object location and movement. This project strengthened skills in serial communication, data parsing, and real-time visualization, and demonstrated the value of combining embedded systems with higher-level software tools for monitoring and analysis.
